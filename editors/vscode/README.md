# Ketsa Language

Official Visual Studio Code extension for the Ketsa Programming Language.

Ketsa provides language support, syntax highlighting, and integrated execution support for `.ketsa` files.

## Features

- `.ketsa` file support
- Syntax highlighting
- Ketsa file icons
- Run Ketsa button
- Integrated Ketsa Output panel
- Automatic Ketsa detection
- Ketsa project support

## Usage

1. Install the Ketsa compiler.
2. Open a `.ketsa` file in VS Code.
3. Press the **Run Ketsa** button.
4. View the output in the **Ketsa Output** panel.
